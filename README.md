# Inkblot
A from-scratch engine/game-like project written in C++26 and Vulkan.
  
Primarily developed as a playground for exploring engine architecture, rendering systems, memory allocation, platform abstraction, and modern C++ design.

## How to Use
This project can be operated via a Makefile that wraps CMake
```bash
# Configure CMake
> make configure

# Build the application without running
> make build

# Build and run the application
> make run

# Build and run the tests
> make test

# Clean the build directory
> make clean
```
